# 🐑 Woolly Petshop API

REST API desarrollada con **Flask** para la gestión de productos y categorías de una tienda de mascotas.

Este proyecto fue desarrollado como parte de un Bootcamp Full Stack y aplica una arquitectura organizada por capas utilizando **Resources**, **Services** y **Models**.

## ✨ Características

- 🔐 Autenticación con JWT.
- 📦 CRUD de productos.
- 🗂️ CRUD de categorías.
- 📈 Endpoint de lógica de negocio para reabastecer stock.
- ✅ Validaciones con Pydantic.
- 📚 Documentación interactiva con Swagger.
- 🐘 Base de datos PostgreSQL.
